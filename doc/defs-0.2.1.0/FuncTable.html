insert :: Text -> Signature -> Handler v -> FuncTable v -> FuncTable v Source #

Insert a new name, signature and handler in the table. If the name and signature pair is already present in the table, the associated handler is replaced with the supplied handler.

union :: FuncTable v -> FuncTable v -> FuncTable v Source #

The expression (union t1 t2) takes the left-biased union of t1 and t2. It prefers t1 when duplicate name and signature pairs are encountered.
