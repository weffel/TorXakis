Sum type

data Sum a Source #

Sum is a parameterized data type. Sum represents a symbolic sum of terms of the type parameter a. The same term can occur multiple times.

sum :: Ord a => Sum a -> Sum a -> Sum a Source #

O(n+m). The sum of two sums.

The implementation uses the efficient hedge-union algorithm. Hedge-union is more efficient on (bigsum sum smallsum).
