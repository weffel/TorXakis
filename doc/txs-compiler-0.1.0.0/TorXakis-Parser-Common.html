Parser for the body of the declaration. The name given as a parameter is the name of the declaration. The location given as parameter is the location in which the declaration name was found.

-> TxsParser a 

Parser for declarations of the form:

XDEF n ::= p ENDEF

Where n is the name of the definition, p is the parser that parses the body of the declaration, and XDEF is the declaration name (e.g. MODELDEF).

-> TxsParser a 

Parser for declarations with parameters of the form: XDEF n params ::= body ENDEF

Where n is the name of the definition, params are the parameters taken by the declaration, body is the body of the declaration, and XDEF is the declaration name (e.g. PROCDEF). The boolean parameter of this function indicates whether 'ENDDEF' is required.
