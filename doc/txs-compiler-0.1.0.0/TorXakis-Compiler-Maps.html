join :: (Ord k, Ord v) => Map k v -> Map v w -> Map k w Source #

Join two maps m0 and m1. A pair '(k, w)' is in the resulting map if and only if there is a key v such that '(k, v)' is in m0 and '(v, w)' is in m1.
